# SafeHaven - Frontend

Este es el frontend de la aplicación SafeHaven, desarrollado con React, Vite y Bootstrap.

---

## Tecnologías usadas

- React 18+
- Vite
- Bootstrap 5
- ESLint + Prettier para calidad de código

---

## Requisitos previos

- Node.js v16 o superior  
- npm v8 o superior  

---

## Instalación

1. Clonar el repositorio (si no lo hiciste aún):

git clone https://github.com/tu-usuario/tu-repo-front.git


2. Instalar dependencias:

npm install

---

## Ejecución en desarrollo

Para iniciar el servidor de desarrollo con hot reload:


npm run dev

Luego abre en tu navegador:


http://localhost:5173


---

## Calidad de código

- Ejecutar linter:

npm run lint

- Ejecutar linter y arreglar problemas automáticos:


npm run lint:fix

- Formatear código con Prettier:


npm run format
